# MMapper Audio Assets

This repository contains audio assets used by [MMapper](https://github.com/MUME/MMapper).

/areas-alternatives contains files for review as to location

Audio was generated using AI at suno.com.
Files were then converted into stems, edited and remixed to suit looping and themes.

In order to create someting that felt correct, Chatgpt was used to create descriptive text prompts of pieces from Howard Shore's soundtrack to the Lord of The Rings Movies.

These were then adapted to specific use cases, and refined as prompts until suitable pieces were generated on Suno for editing.

To use your own music, simply place your own music in the areas folder with the title exactly matching the piece you want to replace.

Restart Mmapper for the changes to take effect.

Mmappeer will automatically loop in game and will cross fade between areas for you.
